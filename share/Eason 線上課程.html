什麼是 Vibe Coding？

這不是傳統的程式課程。我們不教你背誦字典，我們教你當**「指揮官」**。

    你只要會說中文，清楚描述邏輯，AI 就能幫你寫出完美的 Python 代碼。

常見問題

完全沒有程式基礎適合嗎？
非常適合！這堂課的核心就是「Vibe Coding」，我們利用 AI 工具（如 ChatGPT, Cursor）來輔助寫程式。你不需要背誦複雜的語法，只要學會如何正確地向 AI 描述你的需求，就能寫出強大的爬蟲。
需要準備什麼軟體或電腦規格？
一般的 Windows 或 Mac 電腦即可。我們會教你安裝免費的 VS Code 編輯器與 Python 環境。如果你的電腦跑不動，我們也會教你使用雲端開發環境 (Google Colab)。
網站改版了爬蟲失效怎麼辦？
這是爬蟲必經之路！課程中會教你「維護」的技巧。更棒的是，有了 AI，你通常只需要把新的網頁結構複製給 AI，它就能幫你在一分鐘內修復程式碼。
