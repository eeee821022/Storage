按鈕規格設計書 (Mandala Chart 工具)
本文件旨在詳細定義「曼陀羅思考法工具」中所有按鈕的設計規範，以便未來開發時能保持一致的視覺與互動體驗。
1. 總體設計原則
* 字體 (Font Family): * 主要：'Inter', 'Noto Sans TC', sans-serif
* 互動 (Interaction): * 所有可點擊元素在滑鼠懸停 (hover) 時應有視覺回饋 (例如：背景色、陰影變化)。
   * 應使用 transition 屬性使狀態變化流暢 (例如：transition: all 0.2s ease-in-out;)。
2. 主要功能按鈕 (Action Buttons)
這組按鈕用於主要的工具列功能 (匯入、匯出、截圖、清空)。
2.1. 基礎樣式 (.action-button)
這是標準按鈕的基礎樣式。
* 功能: 執行主要操作。
* CSS 規格:
   * display: inline-flex; (使圖示與文字能彈性對齊)
   * align-items: center; (垂直置中)
   * justify-content: center; (水平置中)
   * vertical-align: middle; (確保按鈕之間能完美垂直對齊)
   * background-color: #ffffff; (white)
   * color: #374151; (text-gray-700)
   * font-size: 0.875rem; (14px, text-sm)
   * font-weight: 500; (medium)
   * padding: 0.5rem 1rem; (垂直 8px, 水平 16px)
   * border-radius: 0.5rem; (8px, rounded-lg)
   * border: 1px solid #e5e7eb; (border-gray-200)
   * box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06); (shadow)
   * transition: all 0.2s ease-in-out;
* Hover 狀態 (:hover):
   * box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06); (shadow-md)
   * border-color: #d1d5db; (border-gray-300)
2.2. 圖示 (Icon) 規格 (.action-button svg)
* width: 1.25rem; (20px)
* height: 1.25rem; (20px)
* margin-right: 0.5rem; (8px, 圖示與文字的間距)
* color: #6b7280; (text-gray-500)

4. 右鍵標記選單 (Context Menu Items)
* 功能: 這些是 div 元素，作為右鍵選單中的選項按鈕。
* 容器 CSS (#marker-menu):
   * display: none; (由 JS 切換為 fixed)
   * z-index: 50;
   * background-color: #ffffff; (white)
   * font-size: 0.875rem; (14px, text-sm)
   * border-radius: 0.5rem; (8px, rounded-lg)
   * box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04); (shadow-xl)
   * border: 1px solid #e5e7eb; (border-gray-200)
* 選項按鈕 CSS (div[data-color]):
   * padding: 0.5rem 1rem; (垂直 8px, 水平 16px)
   * cursor: pointer;
* Hover 狀態 (:hover):
   * background-color: #f1f5f9; (hover:bg-gray-100) (此為「清除標記」的範例，其他按鈕有各自的 hover 顏色)
* HTML 編碼 (完整選單):
