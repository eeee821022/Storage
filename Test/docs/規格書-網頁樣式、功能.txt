網頁設計規格書 (Design System - Gray)
本文件旨在詳細記錄「灰色 (Gray)」色系設計規範，包含字型、顏色與組件樣式。此規範可作為一個獨立的設計系統，應用於需要柔和、中性視覺風格的專案。
1. 字型系統 (Typography)
本專案主要字型設定，優先使用 Inter，並以 Noto Sans TC 作為繁體中文的備援。
* 基礎設定 (CSS):
font-family: 'Inter', 'Noto Sans TC', sans-serif;
color: #1f2937; /* text-gray-800 (預設內文色) */

	3. 組件 (Components)
3.1. 主要功能按鈕 (.action-button - Gray)
   * CSS 規格:
   * display: inline-flex;
   * align-items: center;
   * vertical-align: middle;
   * background-color: #ffffff; (white)
   * color: #374151; (text-gray-700)
   * font-size: 0.875rem; (14px)
   * font-weight: 500; (medium)
   * padding: 0.5rem 1rem; (8px 16px)
   * border-radius: 0.5rem; (8px)
   * border: 1px solid #d1d5db; (border-gray-300)
   * box-shadow: 0 1px 3px 0 rgba(0, 0, 0, 0.1), 0 1px 2px 0 rgba(0, 0, 0, 0.06);
   * transition: all 0.2s ease-in-out;
   * Hover 狀態 (:hover):
   * box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06);
   * border-color: #9ca3af; (border-gray-400)
   * 圖示 (Icon) 規格:
   * width: 1.25rem; (20px)
   * height: 1.25rem; (20px)
   * margin-right: 0.5rem; (8px)
   * color: #6b7280; (text-gray-500)
3.2. 說明按鈕 (#help-button)
   * CSS 規格:
   * position: fixed;
   * top: 1.5rem; (24px)
   * right: 1.5rem; (24px)
   * z-index: 50;
   * background-color: #ffffff; (white)
   * padding: 0.75rem; (12px)
   * border-radius: 9999px; (rounded-full)
   * box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05);
   * color: #2563eb; (text-blue-600)
   * transition: background-color 0.15s ease-in-out;
   * Hover 狀態 (:hover):
   * background-color: #eff6ff; (hover:bg-blue-50)
   * 圖示 (Icon) 規格:
   * width: 1.5rem; (24px)
   * height: 1.5rem; (24px)
3.3. 彈出式視窗 (Modal)
   * 遮罩 (.modal-overlay):
   * background-color: rgba(0, 0, 0, 0.5);
   * 內容框 (.modal-content):
   * background-color: #ffffff; (white)
   * border-radius: 0.5rem; (8px)
   * box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04); (shadow-xl)
   * max-height: 80vh;
   * 關閉按鈕 (#close-modal-button):
   * color: #9ca3af; (text-gray-400)
   * hover:color: #4b5563; (hover:text-gray-600)
3.4. 右鍵標記選單 (#marker-menu)
   * 容器:
   * background-color: #ffffff; (white)
   * border-radius: 0.5rem; (8px)
   * border: 1px solid #e5e7eb; (border-gray-200)
   * box-shadow: 0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05); (shadow-lg)
   * 選項:
   * font-size: 0.875rem; (14px)
   * padding: 0.5rem 1rem; (8px 16px)
   * cursor: pointer;
   * 選項 (Hover):
   * hover:bg-gray-100 (清除標記)
   * hover:bg-yellow-100 (黃色)
   * hover:bg-green-100 (綠色)
   * hover:bg-red-100 (紅色)
