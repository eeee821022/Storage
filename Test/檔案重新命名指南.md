# 檔案重新命名指南

為了讓專案更容易管理和維護，建議您將工具檔案重新命名為更簡潔的名稱。

## 📝 建議的重新命名對照表

| 原始檔名 | 建議新檔名 | 說明 |
|---------|-----------|------|
| `1763304193199_網頁-使用者地圖.html` | `customer-journey-map.html` | 顧客旅程地圖工具 |
| `1763304193199_網頁-曼陀羅思考法.html` | `mandala-chart.html` | 曼陀羅思考法工具 |
| `1763304193199_網頁-親和圖法.html` | `affinity-diagram.html` | 親和圖法工具 |

## ⚠️ 重要提醒

如果您決定重新命名檔案，請務必同時修改 `index.html` 中的連結！

### 需要修改的位置

在 `index.html` 中搜尋並替換以下三處：

1. **顧客旅程地圖的連結**
   ```html
   <!-- 修改前 -->
   <a href="1763304193199_網頁-使用者地圖.html" class="primary-button w-full">
   
   <!-- 修改後 -->
   <a href="customer-journey-map.html" class="primary-button w-full">
   ```

2. **曼陀羅思考法的連結**
   ```html
   <!-- 修改前 -->
   <a href="1763304193199_網頁-曼陀羅思考法.html" class="primary-button w-full">
   
   <!-- 修改後 -->
   <a href="mandala-chart.html" class="primary-button w-full">
   ```

3. **親和圖法的連結**
   ```html
   <!-- 修改前 -->
   <a href="1763304193199_網頁-親和圖法.html" class="primary-button w-full">
   
   <!-- 修改後 -->
   <a href="affinity-diagram.html" class="primary-button w-full">
   ```

## 🔧 快速修改方法

### 方法一：使用文字編輯器 (推薦)

1. 在 VS Code、Sublime Text 或其他編輯器中開啟 `index.html`
2. 使用「尋找與取代」功能（通常是 `Ctrl+H` 或 `Cmd+H`）
3. 依序替換三個檔案名稱
4. 儲存檔案

### 方法二：使用命令列 (進階)

在專案資料夾中執行：

**macOS / Linux:**
```bash
# 重新命名檔案
mv "1763304193199_網頁-使用者地圖.html" "customer-journey-map.html"
mv "1763304193199_網頁-曼陀羅思考法.html" "mandala-chart.html"
mv "1763304193199_網頁-親和圖法.html" "affinity-diagram.html"

# 更新 index.html 中的連結
sed -i '' 's/1763304193199_網頁-使用者地圖\.html/customer-journey-map.html/g' index.html
sed -i '' 's/1763304193199_網頁-曼陀羅思考法\.html/mandala-chart.html/g' index.html
sed -i '' 's/1763304193199_網頁-親和圖法\.html/affinity-diagram.html/g' index.html
```

**Windows (PowerShell):**
```powershell
# 重新命名檔案
Rename-Item "1763304193199_網頁-使用者地圖.html" "customer-journey-map.html"
Rename-Item "1763304193199_網頁-曼陀羅思考法.html" "mandala-chart.html"
Rename-Item "1763304193199_網頁-親和圖法.html" "affinity-diagram.html"

# 更新 index.html 中的連結（需手動進行）
```

## ✅ 驗證步驟

完成重新命名後，請執行以下步驟確認：

1. 在瀏覽器中開啟 `index.html`
2. 依序點擊三個「開始使用」按鈕
3. 確認每個工具都能正常開啟
4. 如果出現「找不到頁面」的錯誤，請檢查檔案名稱是否正確
